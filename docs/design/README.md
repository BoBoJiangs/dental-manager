## 当前确认的关键业务结论

- 支持单诊所 + 连锁门诊
- 不做多租户 SaaS
- 集团总部统一患者池，同一患者跨门诊共用主档案
- 完整财务流程
- 患者微信小程序预约
- 会员体系
- 短信通知
- 简单电子签名 / 知情同意书
- 医保 / 发票税控接口预留
- 影像设备先按普通文件上传处理
- PC 端为 Web
- 医生端为 H5 + App
- 患者端为微信小程序
- 支持多门诊/分院
- 医生支持跨门诊排班
- 财务和院长可看全院数据
- 医生仅可查看自己的历史接诊患者和当前主治患者
- 支持牙位图、正畸/种植模板、术前术后照片对比、病例打印模板
